# Triage Standard Prompt

This folder contains the default Triage prompt type for normal maintenance intake and handoff.

## Use This Folder First

Start with [`000_universal.md`](./000_universal.md). It is the canonical/default prompt in this folder and the default entry point for the Triage family.
