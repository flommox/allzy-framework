# Prompt Config Block Template

This folder contains the reusable Prompt Config Block structure used inside operational prompts.

## Use This Folder First

Start with [`prompt_config_block_template.md`](./prompt_config_block_template.md). It is the canonical/default template in this folder.

## What This Folder Contains

- [`prompt_config_block_template.md`](./prompt_config_block_template.md): execution-config structure for prompts.

## Nearby Differences

- A Prompt Config Block controls how a prompt run behaves.
- [`../metadata-frontmatter/metadata_frontmatter_template.md`](../metadata-frontmatter/metadata_frontmatter_template.md) describes an artifact and is not execution config.
- [`../context-package/context_package_template.md`](../context-package/context_package_template.md) packages selected context output after retrieval or planning.
