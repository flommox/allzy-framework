# Yang Templates

This folder contains the reusable Full Yang contract structure.

## Use This Folder First

Start with [`yang-core-module/03_yang_template.md`](yang-core-module/03_yang_template.md). It is the canonical/default Yang template in this repository.

## What This Folder Contains

| Folder | Canonical template | Use for |
|---|---|---|
| [`yang-core-module/`](yang-core-module/) | [`03_yang_template.md`](yang-core-module/03_yang_template.md) | Full Yang Core Module |

## Nearby Differences

- Yang is the deterministic technical execution contract.
- Yin templates in [`../yin/`](../yin/) are for human intent only.
- Combined templates in [`../yin-yang/`](../yin-yang/) are for artifacts that keep both layers together.

Use Full Yang when the work needs schemas, decision rules, invariants, side-effect boundaries, and traceability.

There is no standalone Yang Compact.
