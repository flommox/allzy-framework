# Yin/Yang Templates

This folder contains reusable combined artifacts that keep Yin intent and Yang execution rules in one file.

## Use This Folder First

Choose the scope first:

- Use [`yin-yang-document/04_yin_yang_template.md`](yin-yang-document/04_yin_yang_template.md) for the Full combined document.
- Use [`yin-yang-compact/04_yin_yang_template_compact.md`](yin-yang-compact/04_yin_yang_template_compact.md) for the compact combined document.

The numbered template file in each leaf folder is the canonical structure.

## Contents

| Folder | Canonical template | Use for |
|---|---|---|
| [`yin-yang-document/`](yin-yang-document/) | [`04_yin_yang_template.md`](yin-yang-document/04_yin_yang_template.md) | Full combined Yin/Yang Document |
| [`yin-yang-compact/`](yin-yang-compact/) | [`04_yin_yang_template_compact.md`](yin-yang-compact/04_yin_yang_template_compact.md) | Compact combined Yin/Yang artifact |

## Nearby Differences

- Full Yin/Yang is for one-file specifications that still need complete depth.
- Yin/Yang Compact is for smaller bounded executable work that needs concise intent and concise rules together.
- Yin-only artifacts belong in [`../yin/`](../yin/).
- Yang-only artifacts belong in [`../yang/`](../yang/).

If you need the prompt family that generates specification prompts for these artifacts, use [`../../prompts/Specification/yin-yang-prompt-generator/000_universal.md`](../../prompts/Specification/yin-yang-prompt-generator/000_universal.md).

There is no standalone Yang Compact.
